## REST API란?
- API = Application programming interface
- REST API = Representational state transfer application programming interface 
- 어떤 웹사이트든 개발자들이 작성해둔 API가 있습니다. 해당 API를 사용해서 웹페이지 안의 정보들을 다른 개발자들이 이용하기 쉬운 형태로 받아옵니다.
- 보통은 JSON형태로 반환해줌.
